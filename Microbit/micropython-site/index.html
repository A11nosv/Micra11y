¿Qué es este manual?

Este sitio es una adaptación web del manual MicroPython en el Aula. Cada módulo es una página independiente con ejemplos de código interactivos, ejercicios graduados por nivel y adaptaciones DUA integradas en cada actividad.

No hace falta seguir el orden: cada capítulo es autocontenido. Usa la barra lateral para navegar directamente al tema que necesites.

Cómo usar este sitio

  • Navega por la barra lateral para ir directamente a cualquier capítulo.
  • Cada ejercicio tiene cajas DUA desplegables (🟥🟦🟩) con adaptaciones concretas.
  • Los ejercicios están marcados por nivel: 🟢 Iniciación 🟡 Básico 🟠 Intermedio 🔴 Avanzado
  • Los bloques de código tienen resaltado de sintaxis y nombre de archivo para identificarlos fácilmente.
